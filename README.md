### Built With

- [React Native](https://reactnative.dev/)

- [React Native Paper](https://callstack.github.io/react-native-paper/)

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/PritamG-git/GetLook_Demo
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
